what is HTML?

When you make web page by Hypertext Markup Language(HTML),
you should use tag. There are more than 150 tags.
You can search them on google.

    'strong' is hilight.
    'u' is underline.
    'h1' is appearing topic.
    'br' is creating new line.
    'p' is creating new paragraph. And, you can adjust the line spacing finely like style="margin-top:40px;"...
    'img' is inserting photo. You can bring pictures you want and you can specify the size.
    'li' is creating list. and you can divide them by using "ul"(unordered list).
    'ol' (ordered list) is creating number automatically. It is so comfortable.
    'title' is title of web page.
    All web pages are composed of head, body sections.
    'a href="" target="" title=""' is hyperlink.
